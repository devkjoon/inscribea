# Assets Directory

Place your add-in icons here. Required sizes:

- `icon-16.png` - 16x16 pixels
- `icon-32.png` - 32x32 pixels  
- `icon-64.png` - 64x64 pixels
- `icon-80.png` - 80x80 pixels

You can create simple icons using any image editor, or use online tools like:
- [Favicon Generator](https://favicon.io/)
- [Canva](https://www.canva.com/)

For now, the add-in will work without icons, but Outlook may show placeholder icons.
